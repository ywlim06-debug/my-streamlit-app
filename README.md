# [돌다리]

## 소개
[결정이 어려운 순간, 정답을 주기보다 질문을 통해 생각을 정리하도록 돕는 AI 결정 코칭 앱입니다.
한 화면에 한 질문씩 답하며 진행하고, 마지막에 고민의 핵심과 선택 기준을 요약해줍니다(결론/추천 없음).]
## 주요 기능
- 기능 1. 상황 설정: 선택 주제/상황 설명/원하는 목표를 입력해 세션 시작
- 기능 2. 답변 반영 질문 생성: 이전 답변을 기억해 다음 질문을 더 적절하게 생성(동적 질문)
- 기능 3. 결과 요약 제공: 고민의 핵심 / 선택 기준 / 코칭 메시지로 정리(결론·추천 금지)

## 사용 방법
1. 링크 접속
2. 질문에 답변
3. AI 결과 확인
